# ⚙️ C++ API

The C++ API reference is generated with [Doxygen](https://www.doxygen.nl/) and hosted as a separate site:

**[franky C++ API reference](https://timschneider42.github.io/franky/cpp/)**

The C++ API closely mirrors the [Python API](python.md), though some variable and method names differ because franky
follows [PEP 8](https://peps.python.org/pep-0008/) naming conventions in Python
and [Google naming conventions](https://google.github.io/styleguide/cppguide.html) in C++.
