Detailed Description

template<typename T>
class franky::WaitFreeTripleBuffer< T >

Wait-free, Single-Producer Single-Consumer (SPSC) triple buffer.

This buffer allows a single user thread to safely write data without ever blocking, and a single real-time loop to safely read the latest available data without locking or experiencing torn reads.
